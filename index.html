mumax3
 GPU-accelerated micromagnetism

Home Documentation Download

mumax3 is a GPU-accelerated micromagnetic simulation program developed at Ghent University, DyNaMat group. The software is free and open-source under a GPLv3 license. Its predecessor mumax2 is still available here.

mumax3 offers a speed-up of up to 100x compared to CPU-based simulations. Additionally, it is optimized for low memory use and easily allows over 10 million FD cells.

included physics

  • Landau-Lifshitz micromagnetic formalism
  • Magnetostatic field using FFT-accelerated convolution
  • Heisenberg exchange and Dzyaloshinskii-Moriya interaction
  • Arbitrary inter-region exchange like RKKY coupling
  • Spin-transfer torque (Zhang-Li)
  • Uniaxial and cubic magnetocrystalline anisotropy
  • All material parameters can be fully time- and space-dependent
  • Arbitrary complex excitation (field, current)

extra features:

  • Simulation window can automatically follow a moving domain wall
  • Edge charges can be removed to simulate an infinitely long geometry.

web GUI

mumax3 includes a browser-based user that interface lets you follow a running simulation or modify it on-the-fly, be it on your local machine or remotely.

simple scripting

mumax3 provides simple yet powerful input scripting. E.g., the example applies a time-dependent external field to a uniform magnet (FMR experiment).


setgridsize(128, 32, 4)
setcellsize(5e-9, 5e-9, 5e-9)
Msat = 860e3
Aex  = 13e-12
alpha= 0.2
m=uniform(1, 1, 0)

f := 1e9  // 1GHz
A := 0.01 // 10mT
B_ext = vector(0.1, A*sin(2*pi*f*t), 0)

run(10e-9)

requirements

mumax3 is cross-platform and runs on Linux, Windows and Mac platforms. You need an nVIDIA GPU with compute capability 2.0 or higher, as listed here.

We recommend:
  • GTX 670 (2GB) for very good performance per price
  • GTX 680 (2GB) for better performance, but pricier
  • GTX TITAN (6GB) for very large simulations

Also, you need to install nVIDIA's CUDA toolkit.

Finally you need to use nVIDIA's proprietary graphics driver, which may already be installed on your system.
